## Tau examples

This directory contains checked-in demo assets for package, extension, and events workflows.

- `starter/`: installable package manifest plus referenced components.
- `extensions/`: standalone extension manifest + payload for `--extension-*` commands.
- `events/`: scheduler event definitions and state fixtures for inspect/validate/simulate examples.
- `pacman-tetris/`: self-contained Phaser 3 browser game example that mixes Pac-Man pellet collection with falling tetromino pressure.
- `pacman-tetris-ws/`: a wider Phaser 3 Pac-Man + Tetris sibling example with three ghosts, row-burst stuns, and a distinct arcade HUD.
